# CLI-Tools

A monorepo of small, self-contained CLI tools written in Go and released via GitHub Releases.

## Tools

| Tool | Description |
|------|-------------|
| [unsplash](tools/unsplash/README.md) | Search for photos on Unsplash directly from your terminal |
| [rss](tools/rss/README.md) | Fetch and display RSS/Atom feed items with date filtering and pagination |
| [web2md](tools/web2md/README.md) | Convert a web page to Markdown (bundles [defuddle](https://github.com/kepano/defuddle) binaries) |
